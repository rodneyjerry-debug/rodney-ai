Connect Google Analytics

To show live analytics data, you need to complete a one-time Google Cloud setup:

  1. Go to Google Cloud Console and enable the Google Analytics Data API
  2. Go to Credentials → Create Credentials → OAuth Client ID
  3. Application type: Web application
  4. Add https://rodney-ai.com to Authorized JavaScript origins
  5. Copy the Client ID and paste below
  6. Find your GA4 Property ID: Analytics → Admin → Property Settings → Property ID (numeric)
